# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-21

### Added
- Initial release of ClawBird
- CDP-based browser control
- Playwright integration for advanced actions
- 11 action types (click, type, press, hover, drag, select, fill, resize, wait, upload, download)
- Tab management (list, open, close, focus)
- Screenshot capabilities (full page, element, viewport)
- HTML and ARIA snapshots
- Console log capture
- Network monitoring
- Device emulation
- Geolocation override
- Workflow automation system
- Price monitoring workflow
- News aggregation workflow
- Form filler workflow
- Screenshot documentation workflow
- OpenClaw plugin integration
- Web search skill with fallback
- Comprehensive test suite

### Features
- AI-friendly element references (aria-ref)
- Standalone browser (no extension required)
- Cross-platform support (macOS, Linux, Windows)
- Multiple browser support (Chrome, Brave, Edge)
